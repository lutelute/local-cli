## これは なに？

Local CLI は、あなたの パソコンで うごく AI です。
にほんごで おねがい すると、ファイルを よんだり、かいたり、プログラムを うごかしたり できます。

## つかうのに ひつような もの

- **Python 3.10** いじょう
- **[Ollama](https://ollama.com)** — AI を うごかす アプリ（むりょう）

## はじめかた

### 1. Ollama を いれる

[ollama.com](https://ollama.com) から ダウンロード して、きどう します。

### 2. Local CLI を いれる

```bash
# ダウンロード
git clone https://github.com/lutelute/local-cli.git
cd local-cli

# うごかす
python -m local_cli
```

### 3. つかう

きどう すると `You>` と でます。ここに にほんごで おねがいを かきます。

```
You> hello.txt に「こんにちは」と かいて
```

AI が ファイルを つくって くれます。

## できること

| コマンド | せつめい |
|----------|---------|
| `bash` | シェル コマンドを じっこう |
| `read` | ファイルを よむ |
| `write` | ファイルを つくる |
| `edit` | ファイルを なおす |
| `glob` | ファイルを さがす |
| `grep` | ファイルの なかみを けんさく |

## べんりな コマンド

| コマンド | せつめい |
|----------|---------|
| `/help` | つかいかたを みる |
| `/model <なまえ>` | AI モデルを かえる |
| `/models` | モデルを えらぶ がめん |
| `/status` | いまの じょうたい |
| `/install <モデル>` | モデルを ダウンロード |
| `/update` | さいしんばんに こうしん |
| `/save` | かいわを ほぞん |
| `/exit` | おわる |

## モデルの えらびかた

```bash
# モデルを えらんで きどう
local-cli --select-model

# モデルを していして きどう
local-cli --model qwen3:8b
```

### おすすめ モデル

| モデル | サイズ | とくちょう |
|--------|--------|----------|
| `qwen3:8b` | 5.2 GB | なんでも できる |
| `qwen2.5-coder:7b` | 4.7 GB | プログラミングが とくい |
| `qwen3:0.6b` | 0.5 GB | かるい、テスト よう |

## こうしん（アップデート）

さいしんばんに する ほうほう：

```bash
# コマンドで こうしん
local-cli --update

# または REPL の なかで
/update
```

## デスクトップ アプリ

パソコン アプリ としても つかえます：

```bash
cd desktop
npm install
npm run dev
```
