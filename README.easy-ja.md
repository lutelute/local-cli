## これは なに？

Local CLI は、あなたの パソコンで うごく AI です。
にほんごで おねがい すると、ファイルを よんだり、かいたり、プログラムを うごかしたり できます。

## ダウンロード

### パソコン アプリ（かんたん）

**[GitHub Releases](https://github.com/lutelute/local-cli/releases)** から ダウンロード できます。

| パソコン | ファイル |
|----------|---------|
| Mac | `Local-CLI-x.x.x-universal.dmg` |
| Windows | `Local-CLI-Setup-x.x.x.exe` |
| Linux | `Local-CLI-x.x.x.AppImage` |

ダウンロード した ファイルを ひらいて、インストール します。

> **だいじ:** [Ollama](https://ollama.com) も いれて ください。AI を うごかすのに ひつよう です。

#### Mac で「アプリが こわれています」と でた とき

ターミナルで この コマンドを いれて ください：

```bash
xattr -cr /Applications/Local\ CLI.app
```

または：**システムせってい > プライバシーと セキュリティ > した に スクロール >「このまま ひらく」を クリック**

### コマンドライン（くわしい ひと むけ）

```bash
# ひつような もの: Python 3.10 いじょう, Ollama, Git

# 1. ダウンロード
git clone https://github.com/lutelute/local-cli.git
cd local-cli

# 2. うごかす
python -m local_cli
```

### 3. つかう

きどう すると `You>` と でます。ここに にほんごで おねがいを かきます。

```
You> hello.txt に「こんにちは」と かいて
```

AI が ファイルを つくって くれます。

## できること

| コマンド | せつめい |
|----------|---------|
| `bash` | シェル コマンドを じっこう |
| `read` | ファイルを よむ |
| `write` | ファイルを つくる |
| `edit` | ファイルを なおす |
| `glob` | ファイルを さがす |
| `grep` | ファイルの なかみを けんさく |

## べんりな コマンド

| コマンド | せつめい |
|----------|---------|
| `/help` | つかいかたを みる |
| `/model <なまえ>` | AI モデルを かえる |
| `/models` | モデルを えらぶ がめん |
| `/status` | いまの じょうたい |
| `/install <モデル>` | モデルを ダウンロード |
| `/update` | さいしんばんに こうしん |
| `/save` | かいわを ほぞん |
| `/exit` | おわる |

## モデルの えらびかた

```bash
# モデルを えらんで きどう
local-cli --select-model

# モデルを していして きどう
local-cli --model qwen3:8b
```

### おすすめ モデル

| モデル | サイズ | とくちょう |
|--------|--------|----------|
| `qwen3:8b` | 5.2 GB | なんでも できる |
| `qwen2.5-coder:7b` | 4.7 GB | プログラミングが とくい |
| `qwen3:0.6b` | 0.5 GB | かるい、テスト よう |

## こうしん（アップデート）

きどう した とき、あたらしい バージョンが あると おしらせ が でます。

**パソコン アプリ:** がめんの うえに あおい バーが でたら「Install update」を おして ください。

**コマンドライン:**

```bash
# コマンドで こうしん
local-cli --update

# または つかっている ときに
/update
```

## デスクトップ アプリを じぶんで ビルド

くわしい ひと むけ。ふつうは GitHub Releases から ダウンロード で じゅうぶん です。

```bash
cd desktop
npm install
npm run build:mac    # Mac よう
npm run build:win    # Windows よう
npm run build:linux  # Linux よう
```
